# 프로젝트 주제
LLM API를 활용한 학습 보조 챗봇 개발
(문서 요약, 질문 응답, 문제 생성, 자료 추천 기능을 포함한 챗봇 프로토타입 제작)

# 프로젝트 제작배경
## 과제 배경
ChatGPT, Claude와 같은 대규모 언어모델(LLM)의 등장은 교육·업무·연구 현장에서 문서를 빠르게 이해하고 활용할 수 있는 가능성을 열었습니다. 그러나 단순 대화형 사용에서 벗어나 특정 목적(학습 지원)에 맞는 챗봇 서비스 설계 경험은 학생들에게 필요합니다.

## 목적
교육생들이 LLM API를 직접 연동하고 프롬프트를 설계해봄으로써, 실제 서비스가 동작하는 과정을 경험하게 합니다.

## 해결하고자 하는 문제
- 방대한 문서를 짧은 시간에 이해하기 어렵다.
- 학습자 수준에 맞는 문제/과제를 자동으로 만들기 어렵다.
- 개별 학습자에게 적합한 자료 추천이 부족하다.

## 기대 효과
- LLM API를 활용한 요약, Q&A, 문제 생성, 자료 추천 챗봇을 직접 만들어보며 실무형 역량 습득
- 서비스 기획 → 개발 → 시연의 전 과정을 경험하며 포트폴리오로 활용 가능
- 팀 단위 협업과 발표를 통해 문제 해결 능력 강화

# 프로젝트 달성목표
## 기획 목표
- 챗봇 서비스 시나리오와 사용자 여정을 설계한다.
- 요약, Q&A, 문제 생성, 자료 추천 등 기능별 요구사항을 정의한다.
- 프롬프트 엔지니어링 전략(출력 포맷, 조건 제시)을 기획한다.

## 개발 목표
- OpenAI/Claude/Llama 등 LLM API를 1종 이상 연동하여 챗봇을 구현한다.
- 텍스트 입력 기반으로 요약, Q&A, 문제 생성, 자료 추천 기능을 순차적으로 개발한다.
- Streamlit/간단한 웹 UI를 통해 사용자 인터페이스를 제공한다.
- 완성된 기능을 통합해 학습 지원 챗봇 MVP를 제작하고, 최종 시연 및 발표를 진행한다.

이제 우리팀이 선정한 이번 프로젝트 다음과 같아
# 프로젝트명
학점마스터(Grade Masters)
# 프로젝트 설명
pdf 를 입력받아 내용을 요약해주는 챗봇 서비스
# 대상
중간고사 기말고사를 앞둔 대학생
# 주요기능
- pdf 를 입력받아 내용을 요약해주는 기능
- 문제 은행
  - 요약 내용을 바탕으로 주관식 예상 문제를 생성해주는 기능
  - 생성된 문제를 출제하고 채점하는 기능
  - 채점 결과를 바탕으로 오답노트를 만들고 관련 내용을 정리해서 보여주는 기능
